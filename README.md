# Anipod
유기된 반려동물의 임시 보호 및 봉사활동을 장려하고 보호소와 연결해주는 플렛폼

## 기대 효과
- 임시 보호를 통해 유기된 동물의 위생 및 건강을 호전시키고 안락사를 늦추거나 예방
- 반려동물을 사고 파는 펫 시장의 규모를 상대적으로 줄여 비 윤리적인 문제를 감소
- 플랫폼을 통해 사람들에게 잘 알려지지 않았던 유기된 동물들을 효과적으로 홍보하고 분양을 장려
